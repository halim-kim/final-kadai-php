## ②課題内容（どんな作品か）

- 恋愛に関する10問の質問に答えてみるサイトです。
- 自分のアカウント登録だけでなく、マイページでパートナーも登録でき互いの答えを比較することができます。
- アカウントはIDとPWのみで作れます。
- UIはちょっと派手なので、落ち着いたものをLaravelで作ってみたいです。
